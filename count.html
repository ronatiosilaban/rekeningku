Data Structure And Algorithm Test

Dodi mempunyai uang coin sejumlah 10, 5, 2, 3, 6, 7 dan ingin membeli barang seharga 23 coin dengan biaya transaksi 0.5 coin per tiap coin, sebutkan coin yang di perlukan berdasarkan biaya transaksi termurah, sebutkan sisa coin yang dimiliki dan biaya transaksi nya.**

Tulis code dengan bahasa pemrograman node js / typescript / go beserta unit test nya. Nilai plus jika di buat sebagai aplikasi rest api dengan database.

Example

Input: [10, 5, 2, 3, 6, 7]

Total Transaksi: 23

Fee Per Coin: 0.5
